# **App Name**: Claritas Copilot

## Core Features:

- AI Financial Copilot: AI-powered insights provide proactive and personalized financial advice, delivered as insights.
- Dashboard View: Dashboard offering a high-level overview of the user's financial status, including net worth, spending habits, upcoming bills, and progress towards financial goals.
- Spending Analysis: Spending analysis tools offer detailed breakdowns of expenses by category, to help users identify areas where they can save money.
- Goal Setting: Goal setting functionality enables users to set financial goals and track their progress over time. Users can set goals and track progress using visual progress indicators.
- AI Chatbot: AI-powered chatbot for answering user questions and providing personalized financial advice. The tool uses natural language understanding.

## Style Guidelines:

- Primary color: Dark blue (#1A237E), reflecting the core concepts of trust and clarity in the financial app. Dark shades suit data presentation well.
- Background color: Light gray (#F5F5F5), ensuring a clean and modern interface that avoids distraction.
- Accent color: Soft coral (#FF7043) is employed for highlights and calls to action. Coral is analogous to blue (about 30 degrees away on the color wheel), but its contrast in brightness and saturation helps guide the user.
- Body and headline font: 'Inter', a sans-serif font, selected for its modern, clean appearance that provides great readability.
- Consistent and modern icon set to provide visual cues and enhance user experience.
- Clean and intuitive layout with a focus on readability and ease of navigation, allowing users to access key information and features effortlessly.
- Subtle animations and transitions used to create a fluid and engaging user experience, enhancing usability.
